Shirt Export

  1. Open the production file with Photoshop
  2. Select the main layer on the layers panel
  3. Press M to select the Rectangular Marquee Tool
  4. Select the shirt area in the bottom
  5. Press Ctrl + J (cmd + J in mac) This will create a new layer
  6. Right click on the new layer in the layers panel and select "Export As..."
  7. Make sure the format is PNG, and press Export.
  8. Select the Export Location and press Export again
  9. Open the exported file with Photoshop
  10. Open the shirt-guidelines.psd
  11. Press V and make sure the Auto-Select is turned off.
  12. Drag the exported file to the shirt-guidelines.psd
  13. Make the file edges match the guidelines
  14. Turn off the guidelines layer
  15. Press "Ctrl + Alt + Shift + W" to export the whole canvas
